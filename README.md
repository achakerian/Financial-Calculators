# Financial Calculators

A mobile-first financial calculator platform for Australian tax residents, built with React, TypeScript, and Tailwind CSS.

## Features

- **Pay & Tax Calculator** - Income tax, HELP/HECS repayments, Medicare levy calculations
- **Loan Calculator** - Mortgage and personal loan comparisons
- **Borrowing Power Calculator** - Estimate borrowing capacity
- **Superannuation Calculator** - Retirement savings projections

## Tech Stack

- React 18.3 + TypeScript 5.5
- Vite 5.2
- Tailwind CSS 3.4
- React Router 7.11
- Recharts 2.10
- pnpm workspaces (monorepo)

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Run tests
pnpm test

# Build for production
pnpm build
```
